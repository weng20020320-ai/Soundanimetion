# Screenshots for clearmika.com

主页 / `apparatus.json` 会读取这个目录下的图。

## 命名规范

| 文件名 | 用途 | 尺寸 | 备注 |
|---|---|---|---|
| `cover.png` | Apparatus 卡片背景 / 详情弹窗主图 | **1920×1080**（16:9） | 必需。主截图。 |
| `detail-01.png` | 详情弹窗第二张 | 1920×1080 或 1080×1920 | 可选 |
| `detail-02.png` | 详情弹窗第三张 | 1920×1080 或 1080×1920 | 可选 |
| `detail-03.png` | 详情弹窗第四张 | 1920×1080 或 1080×1920 | 可选 |

## 截图建议

参考 clearmika.com 主页气质（深空海军蓝、安静、克制）：

- 程序窗口处于**深色背景**下（默认 `bgColor: '#0b0d12'` 即可）
- 加载一段能让画面"动"起来的音频，调出**最克制的渐变**（推荐 `midnight-violet`、`twilight-glacier`、`carbon-fade` 这类雾系/冷色）
- **不要**用过分鲜艳的霓虹色或彩虹色，那会和主页气质冲突
- 推荐预设：`wave-line`（流光波形线，安静）、`area-spectrum`（填充频谱，干净）、`shader-flow`（频谱流体，抽象）
- 截图时把 Tweakpane 参数面板**隐藏或最小化**，让画面成为主体
- 在 1920×1080 全屏或近似全屏窗口下截图（不要带浏览器 / Windows 标题栏）

## 操作流程

1. 启动程序：双击 `サウンド可視化.bat` 或 `npm run dev`
2. 加载一段音频，让程序播放到比较有视觉张力的位置
3. 暂停播放，**点击界面顶部的「快照」按钮** —— 这是项目自带的功能，会保存当前 canvas 的真实像素到 PNG
4. 把保存下来的 PNG 重命名为 `cover.png` 放到本目录

## 占位符

正式截图就位之前，`cover.png` 可以缺席。`apparatus.json` 字段已经写好引用路径，主页那边读到 404 会自动 fallback 到纯色卡片（不会崩）。
